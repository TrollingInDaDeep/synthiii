Midi Mapping AM synth (not affected by channel filter or midi cable connected to am synth)
Set the mapping accordingly through MIDI learn via Web UI.
You can't map the dropdowns via webUI, but you can map it in the .ttl file. Go to the control and add following:
```
    midi:binding [
        midi:channel 12 ;
        midi:controllerNumber 8 ;
        lv2:minimum 0.000000 ;
        lv2:maximum 4.000000 ;
        a midi:Controller ;
    ] ;
```
if you want to map the waveform selection of OSC1, add the code like here. it will map to channel 12, CC 8. in your midi controller firmware and here in the ttl file, set the min and max cc value according to how many dropdown items there are (0 based)

```
<amsynth/osc1_waveform>
    ingen:value 2.000000 ;
    midi:binding [
        midi:channel 12 ;
        midi:controllerNumber 8 ;
        lv2:minimum 0.000000 ;
        lv2:maximum 4.000000 ;
        a midi:Controller ;
    ] ;
    a lv2:ControlPort ,
        lv2:InputPort .
```
